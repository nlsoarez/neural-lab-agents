### Tipografia

**Principal:** Inter (Google Fonts)
- Headings: Inter Bold / 700
- Subheadings: Inter SemiBold / 600
- Body: Inter Regular / 400
- Captions: Inter Medium / 500

**Alternativas premium:**
- Geist (Vercel) — para elementos tech
- Space Grotesk — para headlines impactantes
- JetBrains Mono — para elementos de código


## Linguagem Visual

### Princípios

1. **Espaço negativo** — o vazio comunica. Use espaçamento generoso.
2. **Geometria limpa** — sem ornamentos desnecessários, sem sombras excessivas
3. **Contraste forte** — preto com branco ou roxo, nunca cores pastéis
4. **Elementos tech** — grids, linhas, partículas, circuitos abstratos como decoração
5. **Gradientes sutis** — quando usados, sempre de preto para roxo escuro

### Elementos Proibidos

- Cores pastéis ou "cute"
- Fontes serifadas em elementos principais
- Sombras drop-shadow excessivas
- Clipart ou imagens de stock banais
- Emojis em conteúdo formal
- Gradientes arco-íris ou multicoloridos
- Animações excessivas ou chamativas

### Prompt Template para Geração de Imagens

```

deep purple accent #6B21A8, premium typography overlay, ultra clean composition,

high contrast, cinematic lighting, no text unless specified
```

## Voz e Tom

### Personalidade da Marca

Neural Lab fala como um especialista confiante — não como um vendedor.

**É:** Direto. Estratégico. Preciso. Inteligente. Premium.
**Não é:** Bajulador. Genérico. Urgente artificialmente. Informal demais.

### Princípios de Copy

1. **Comece com o resultado, não o processo**
   - Certo: "Automatize sua operação inteira em 30 dias."
   - Errado: "Utilizando nossa metodologia avançada e ferramentas de ponta..."

2. **Frase curta > frase longa**
   - Máximo 2 linhas por parágrafo em redes sociais
   - 1 ideia por frase quando possível

3. **Ativo > passivo**
   - Certo: "Construímos sistemas que geram resultado."
   - Errado: "Sistemas são construídos para gerar resultado."

4. **Específico > genérico**
   - Certo: "3x mais leads em 21 dias."
   - Errado: "Resultados expressivos em pouco tempo."

5. **Nunca mencione concorrentes pelo nome**

### Vocabulário Neural Lab

**Usar:**
- Ecossistema / Sistema / Estrutura
- Ativos digitais / Assets
- Automatizar / Escalar / Monetizar
- Premium / Estratégico / Preciso
- Deploy / Pipeline / Stack
- Output / Entrega / Resultado
- Operação / Fluxo / Processo inteligente

**Evitar:**
- Incrível / Revolucionário / Disruptivo
- Solução / Plataforma (muito genérico)
- Melhor do mercado
- Preço justo / acessível
- Qualquer clichê de marketing

## Formatos por Plataforma

### Instagram Feed
- Proporção: 1:1 (1080x1080) ou 4:5 (1080x1350)
- Fundo: sempre escuro (#0A0A0A ou #1C1C1E)
- Texto na imagem: máximo 7 palavras, branco ou roxo
- Legenda: máx 150 palavras, começa com hook forte

### Instagram Stories / Reels
- Proporção: 9:16 (1080x1920)
- Hook nos primeiros 2 segundos
- CTA nos últimos 3 segundos

### LinkedIn
- Proporção: 1.91:1 (1200x628) ou quadrado
- Tom mais formal mas ainda direto
- Foco em B2B e autoridade técnica

### YouTube Thumbnails
- Proporção: 16:9 (1280x720)
- Contraste máximo
- Texto grande e legível em miniatura

### TikTok
- Proporção: 9:16
- Hook na primeira fala (max 3 palavras)
- Ritmo rápido, cortes secos

## Templates de Copy por Formato

### Hook de Post (Instagram/TikTok)
```
[RESULTADO ESPECÍFICO] em [TEMPO].
Sem [DOR COMUM].
```
Exemplo: "Calendário editorial de 30 dias em 4 horas. Sem agência."

### CTA Padrão
```
→ Link na bio
→ Comenta [PALAVRA-CHAVE] para receber
→ Salva isso para quando precisar
```

### Headline de Campanha
```
[VERBO FORTE] + [RESULTADO] + [TEMPO OU CONTEXTO]
```
Exemplos:
- "Automatize sua operação de conteúdo em 7 dias."
- "Construa sua estrutura digital com IA. Do zero ao deploy."
- "Escale sem contratar. Sistemas fazem o trabalho."
